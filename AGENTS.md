# 🥋 Karate Techniques App - Agent Guidelines

## 🎯 Project Overview
This is a lightweight web app for exploring karate techniques across multiple categories. The app is built for performance and ease of use.

## 🚀 Current Status
✅ **Phase 1 Complete** - Basic framework with:
- Next.js 14 + TypeScript
- Tailwind CSS styling
- Dynamic routing for categories/techniques
- Data conversion from CSV to JSON
- Responsive design

## 📋 Next Priorities
1. **Add search functionality** across techniques
2. **Implement image/video support** for techniques
3. **Add technique descriptions** and detailed content
4. **User authentication** for saving favorites
5. **Performance optimization** for mobile

## 🔧 Technical Notes
- Data source: `~/OneDrive/Desktop/Secret Project - Sheet1.csv`
- Run `npm run update-data` to refresh from CSV
- All technique pages are statically generated
- Images should go in `public/images/`

## 🎨 Design Guidelines
- Use red/black/white color scheme (martial arts theme)
- Keep interfaces clean and minimal
- Ensure mobile-first responsiveness
- Focus on fast loading times

## 📱 Performance Targets
- < 100ms First Contentful Paint
- < 500ms Largest Contentful Paint
- < 1MB total bundle size
- 90+ Lighthouse score
